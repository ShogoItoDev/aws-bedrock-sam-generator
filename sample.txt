1. アプリケーションには、Python言語で記述された単一のAWS Lambdaファンクションが含まれている必要があります。
2. Lambdaファンクションは、AWS API Gateway HTTP APIエンドポイントによってトリガーされる必要があります。
3. Lambdaファンクションは、AWS DynamoDBテーブルからデータを取得する必要があります。
4. DynamoDBテーブルには、以下のスキーマでユーザーデータが格納されている必要があります:
   - userId (パーティションキー、文字列)
   - name (文字列)
   - email (文字列)
5. Lambdaファンクションは、DynamoDBテーブルから取得したユーザーデータをJSONレスポンスとして返す必要があります。
6. アプリケーションは、us-east-1リージョンにデプロイされる必要があります。

これらの要件を満たすために必要なリソースと設定を定義したAWS SAM YAMLファイルを生成してください。
